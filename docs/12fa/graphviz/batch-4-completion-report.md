## Usage Instructions

### Rendering Diagrams
```bash
# Render single diagram to PNG
dot -Tpng seo-specialist-process.dot -o seo-specialist-process.png

# Render all diagrams in batch
for file in *.dot; do
  dot -Tpng "$file" -o "${file%.dot}.png"
done

# Alternative formats
dot -Tsvg agent-name-process.dot -o agent-name-process.svg  # SVG
dot -Tpdf agent-name-process.dot -o agent-name-process.pdf  # PDF
```

### Validation
```bash
# Check syntax
dot -v -Tpng agent-name-process.dot -o /dev/null

# Validate all files
for file in *.dot; do
  echo "Validating $file..."
  dot -v -Tpng "$file" -o /dev/null
done
```
