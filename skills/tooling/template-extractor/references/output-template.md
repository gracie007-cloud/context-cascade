# Output Template (Template Extractor)

Include these sections in every delivery:

1. **Summary** – artifact type, scope, and intended reuse.
2. **Templates** – files/snippets with placeholders and parameter tables.
3. **Provenance** – source paths/lines and licenses.
4. **Usage Notes** – setup, compatibility, and example invocation.
5. **Validation** – checks performed and results.
6. **Risks/Follow-ups** – gaps to address.
7. **Confidence** – `Confidence: X.XX (ceiling: TYPE Y.YY)` and memory key used.
