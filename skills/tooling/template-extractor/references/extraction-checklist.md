# Extraction Checklist

Use before delivering templates; align with Prompt Architect/Skill Forge guardrails.

- Scope and artifact type confirmed (prompt/code/doc/config).
- Source paths/lines captured; no unverified content.
- Placeholders and parameters documented with defaults/examples.
- Constraints and dependencies noted (versions, environment, licensing).
- Validation performed (lint/tests/manual compare) and results recorded.
- Memory tagged: `skills/tooling/template-extractor/{project}/{timestamp}`.
- Confidence ceiling stated: `Confidence: X.XX (ceiling: TYPE Y.YY)`.
